# Простой AI Бот для Twitter (X)

[🇷🇺 Читать на русском](README_ru.md) | [🇬🇧 English](README.md)

Автономный бот для Twitter (X) на Python, который исследует заданную нишу, курирует релевантные новости, генерирует твиты с помощью больших языковых моделей (LLM) и публикует их. Управление ботом полностью осуществляется через удобный интерфейс в Telegram с системой очередей и ручного подтверждения.

## Ключевые функции

- **Двухэтапный конвейер LLM:**
  1. **Кураторство:** Бот парсит десятки RSS лент и сабреддитов, отсеивая рекламу, реферальные ссылки и спам. LLM выступает в роли главного редактора, выбирая один самый интересный и вирусный пост.
  2. **Генерация:** Пишет мощный, оптимизированный для X твит (до 250 символов) с правильным форматированием без клишированных хештегов.
- **Защита от повторений:** Ведет историю `post_history.json` с последними опубликованными твитами. Эта история передается нейросети, чтобы бот не повторял одни и те же темы и фразы изо дня в день.
- **Панель управления в Telegram:** Проверяй черновики через систему очередей. Ты можешь одобрить, отклонить или нажать "Переписать", чтобы получить свежий вариант твита прямо в Telegram.
- **Расписание и Кулдауны:** 
  - Устанавливай рандомизированные интервалы (например, от 1 до 3 часов). Бот будет выбирать случайное время, имитируя живого человека.
  - После публикации бот переходит в строгий режим ожидания (Cooldown), чтобы избежать лимитов API и сэкономить токены LLM.
- **Поддержка любых LLM:** Работает с любым API, совместимым с OpenAI (OpenAI, Anthropic через OpenRouter, DeepSeek или локальные Ollama).
- **Режимы Авто / Ручной:** Позволь боту автоматически находить источники по твоей Нише, или задай свой строгий список RSS и Reddit вручную.
- **Авто-постинг:** Устал проверять вручную? Включи `/noaccept`, и бот будет работать полностью автономно, публикуя лучшие новости по своему расписанию.
- **Двуязычность:** Интерфейс и генерация твитов полностью поддерживают Английский (`en`) и Русский (`ru`) языки. Язык меняется командой `/set_lang`.

## Требования

- Python 3.9+
- Аккаунт Twitter Developer (Consumer и Access токены)
- Токен Telegram бота (от BotFather) и твой Telegram Chat ID
- API ключ от любого поддерживаемого провайдера LLM (OpenAI, OpenRouter, NVIDIA, DeepSeek и т.д.)
- *(Опционально)* API ключ Google Gemini: Используется исключительно для чтения и анализа графиков/картинок из новостных статей.

## Установка

1. **Клонируй репозиторий:**
   ```bash
   git clone git@github.com:ivanchik-byte/Simple-Twitter-X-AIbot.git
   cd Simple-Twitter-X-AIbot
   ```

2. **Создай виртуальное окружение:**
   ```bash
   python -m venv venv
   source venv/bin/activate  # Для Windows: venv\Scripts\activate
   ```

3. **Установи зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

## Настройка

1. **Переменные окружения:**
   Скопируй пример файла с переменными:
   ```bash
   cp .env.example .env
   ```
   Открой `.env` и заполни данные:
   - `LLM_API_KEY`: Твой ключ для генерации текста.
   - `GEMINI_API_KEY`: Твой ключ Gemini (нужен для парсинга картинок).
   - Ключи Twitter (`TWITTER_CONSUMER_KEY` и т.д.)
   - Ключи Telegram (`TELEGRAM_BOT_TOKEN`, `TELEGRAM_CHAT_ID`)

2. **Настройки Бота (`config.yaml`):**
   ```yaml
   bot:
     language: "ru" # "en" или "ru"
     niche: "Искусственный Интеллект"
     tone_of_voice: "Экспертный блогер, профессионально и кратко"
     mode: "auto"
     interval: [3600, 10800] # Случайный интервал от 1 до 3 часов

   llm:
     base_url: "https://integrate.api.nvidia.com/v1"
     model_name: "deepseek-ai/deepseek-v4-pro"
   ```
   - **Language:** Язык интерфейса и постов.
   - **Niche & Tone:** Укажи тему, о которой пишет бот, и его стиль.
   - **Mode:** `auto` (сам ищет источники) или `manual` (строго по списку).
   - **LLM:** Настрой эндпоинт, изменив `base_url` и `model_name` (например, `https://api.openai.com/v1` и `gpt-4o`).

## Команды Telegram и Использование

Запусти бота:
```bash
python main.py
```

Открой своего бота в Telegram. Ты можешь использовать встроенную клавиатуру или следующие команды:

- `/start` или `/help` - Показать меню и список команд.
- `/status` - Статус бота, таймеры кулдауна, статистика и расписание.
- `/parse` (или `/force`) - Принудительно запустить поиск свежих новостей.
- `/queue` - Просмотр постов, ожидающих твоего одобрения.
- `/history` - Просмотр последних 5 опубликованных твитов.
- `/noaccept` - Включить/Выключить режим Авто-постинга.
- `/set_lang <en|ru>` - Переключить язык бота.
- `/set_interval <min> [max]` - Установить интервалы проверок (например, `/set_interval 1h 3h`).
- `/set_niche <name>` - Изменить целевую нишу на лету.
- `/set_mode <auto|manual>` - Переключить режим поиска источников.
- `/set_tone <tone>` - Обновить тон (стиль) гострайтера.
- `/add_rss <url>` / `/add_sub <name>` - Добавить кастомные источники (работает в ручном режиме).

## Заметки по Архитектуре

- **Асинхронность:** Все сетевые запросы к LLM и парсинг RSS обернуты в `asyncio.to_thread`, чтобы интерфейс Telegram оставался мгновенно отзывчивым и не зависал.
- **База данных SQLite:** База использует режим Write-Ahead Logging (WAL) для безопасного параллельного доступа, в основном отслеживая опубликованные URL для предотвращения дубликатов.
- **Fail-Safes (Защита от сбоев):** Включает умное экранирование HTML для обхода лимитов Telegram, точную обрезку текста для X (Twitter) с учетом границ предложений, и автоматический backoff (паузы) при получении ошибки 429 Rate Limit от Reddit.
