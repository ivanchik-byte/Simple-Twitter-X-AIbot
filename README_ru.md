# Простой AI Бот для Twitter (X)

[🇷🇺 Читать на русском](README_ru.md) | [🇬🇧 English](README.md)

Автономный Python-бот для Twitter (X): сам исследует нишу, отбирает лучшие новости, пишет твиты через нейросеть и публикует их. Управляешь всем через Telegram-бота — одобряй, отклоняй или проси переписать черновик, не заходя в код.

## Ключевые функции

- **Двухэтапный конвейер LLM:**
  1. **Кураторство:** Парсит RSS-ленты и сабреддиты, отсеивает рекламу и спам. Нейросеть выбирает один самый сочный пост из пачки.
  2. **Генерация:** Пишет ёмкий твит (до 250 символов) без унылых хештегов — только суть.
- **Память без повторов:** Ведёт логу `post_history.json` с последними твитами. Передаём её LLM, чтобы бот не зацикливался на одних и тех же темах.
- **Пульт в Telegram:** Черновики приходят в очередь. Жми «Опубликовать», «Отклонить» или «Переписать» — всё прямо из мессенджера.
- **Расписание и паузы:**
  - Случайный интервал между проверками (например, от 1 до 3 часов) — чтобы выглядело естественно.
  - После публикации бот уходит в «спячку» (cooldown), обходя лимиты API и экономя токены.
- **Любая LLM — не проблема:** Работает со всем, что дружит с OpenAI API: OpenAI, Anthropic (через OpenRouter), DeepSeek или локальный Ollama.
- **Авто / Ручной режим:** Либо бот сам находит RSS и сабреддиты по твоей нише, либо работает строго по твоему списку.
- **Режим «Забросил и забыл»:** Включи `/noaccept` — бот публикует лучшие новости сам, без твоего одобрения.
- **Два языка из коробки:** Интерфейс и твиты на английском (`en`) или русском (`ru`). Переключается командой `/set_lang`.

## Что нужно

- Python 3.9+
- Twitter Developer Account (Consumer и Access токены)
- Токен Telegram-бота (от BotFather) и твой Chat ID
- API-ключ от любого поддерживаемого LLM-провайдера (OpenAI, OpenRouter, NVIDIA, DeepSeek и т.д.)
- *(Опционально)* Google Gemini API — только для распознавания картинок в новостях.

## Установка

1. **Клонируй репозиторий:**
   ```bash
   git clone git@github.com:ivanchik-byte/Simple-Twitter-X-AIbot.git
   cd Simple-Twitter-X-AIbot
   ```

2. **Создай виртуальное окружение:**
   ```bash
   python -m venv venv
   source venv/bin/activate  # Для Windows: venv\Scripts\activate
   ```

3. **Установи зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

## Настройка

1. **Переменные окружения:**
   Скопируй пример:
   ```bash
   cp .env.example .env
   ```
   Открой `.env` и впиши свои ключи:
   - `LLM_API_KEY` — для генерации текста
   - `GEMINI_API_KEY` — для распознавания картинок (необязательно)
   - Ключи Twitter (`TWITTER_CONSUMER_KEY` и т.д.)
   - Ключи Telegram (`TELEGRAM_BOT_TOKEN`, `TELEGRAM_CHAT_ID`)

2. **Настройки бота (`config.yaml`):**
   ```yaml
   bot:
     language: "ru"              # "en" или "ru"
     niche: "Искусственный Интеллект"
     tone_of_voice: "Экспертный блогер, профессионально и кратко"
     mode: "auto"
     interval: [3600, 10800]     # Случайный интервал от 1 до 3 часов

   llm:
     base_url: "https://integrate.api.nvidia.com/v1"
     model_name: "deepseek-ai/deepseek-v4-pro"
   ```
   - **Language:** Язык интерфейса и твитов.
   - **Niche & Tone:** О чём бот пишет и как он «звучит».
   - **Mode:** `auto` (сам ищет источники) или `manual` (твой список).
   - **LLM:** Укажи `base_url` и `model_name` (например, `https://api.openai.com/v1` и `gpt-4o`).

## Команды Telegram

Запусти бота:
```bash
python main.py
```

Открой бота в Telegram. Работает встроенная клавиатура и текстовые команды:

- `/start` или `/help` — Показать меню и список команд.
- `/status` — Здоровье бота, таймеры кулдауна, статистика, расписание.
- `/parse` (или `/force`) — Принудительно запустить поиск новостей.
- `/queue` — Посты, ожидающие твоего одобрения.
- `/history` — Последние 5 опубликованных твитов.
- `/noaccept` — Включить / выключить авто-постинг.
- `/set_lang <en|ru>` — Сменить язык на лету.
- `/set_interval <min> [max]` — Задать интервал (например, `/set_interval 1h 3h`).
- `/set_niche <name>` — Поменять нишу.
- `/set_mode <auto|manual>` — Переключить режим поиска источников.
- `/set_tone <tone>` — Обновить стиль текстов.
- `/add_rss <url>` / `/add_sub <name>` — Добавить свои источники (работает в ручном режиме).

## Архитектура

- **Асинхронность:** LLM-запросы и парсинг RSS крутятся в `asyncio.to_thread`, так что Telegram-интерфейс не зависает.
- **SQLite в качестве базы:** Использует WAL-режим для безопасного параллельного доступа. Хранит опубликованные URL, чтобы не постить дубликаты.
- **Защита от сбоев:** Экранирование HTML для лимитов Telegram, умная обрезка текста под X (Twitter), автоматические паузы при ошибке 429 от Reddit.
