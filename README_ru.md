[![Python Version](https://img.shields.io/badge/python-3.9+-blue.svg)](https://www.python.org/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

[🇷🇺 Читать на русском](README_ru.md) | [🇬🇧 English](README.md)

Автономный ассистент на Python для работы с Twitter (X), который самостоятельно исследует заданную нишу, находит и анализирует новости из RSS-лент и Reddit, генерирует привлекательные посты с помощью LLM-моделей и управляется с помощью интерактивной панели в Telegram.

## Ключевые возможности

- **Двухэтапный LLM-конвейер:**
  - **Кураторство:** Сканирует RSS и сабреддиты, отфильтровывая рекламный контент, кликбейты, партнерские (аффилиатные) ссылки и спам. LLM автоматически отбирает самую ценную новость из пачки.
  - **Генерация:** Пишет ёмкие, структурированные посты под формат X с цепляющим началом (крючком) без засилья хэштегов. Поддерживает лимиты до 280 символов (для обычных аккаунтов) и до 25 000 символов (для X Premium).
- **Защита от самоповторов:** Сохраняет историю опубликованных материалов в `post_history.json` и передает ее в контекст LLM, исключая повторение тем, формулировок и шаблонов.
- **Интерактивный пульт в Telegram:** Проверяй публикации перед отправкой. Одобряй, отклоняй или проси ИИ перефразировать твит в один клик.
- **Органическое расписание:** Проверки выполняются со случайными интервалами (например, раз в 1-3 часа), что имитирует поведение реального человека и защищает аккаунт.
- **Автоматические паузы (Cooldown):** Бот временно засыпает после каждой публикации, чтобы экономно расходовать API-токены и избегать лимитов платформы X.
- **Совместимость с любыми API:** Поддерживает любые OpenAI-совместимые эндпоинты (OpenAI, Anthropic через OpenRouter, DeepSeek, NVIDIA API, локальный Ollama).
- **Режимы Auto & Manual:** Бот может сам искать популярные источники по заданной нише или работать по жестко заданному вами списку сайтов.
- **Полная автономия:** Режим `/noaccept` отключает необходимость ручного подтверждения — бот начнет постить лучшие новости полностью самостоятельно.
- **Двуязычный интерфейс:** Полная поддержка английского (`en`) и русского (`ru`) языков для управления, кнопок и генерации постов.

## Требования

- Python 3.9 или выше
- Аккаунт Twitter Developer (токены Consumer и Access)
- Токен Telegram-бота (от [@BotFather](https://t.me/BotFather)) и ваш личный Telegram Chat ID
- API-ключ любого LLM-провайдера (OpenAI, DeepSeek, OpenRouter и т.д.)
- *(Опционально)* API-ключ Google Gemini (требуется исключительно для анализа изображений и графиков в новостных статьях)

## Установка

1. **Клонируйте репозиторий:**
   ```bash
   git clone git@github.com:ivanchik-byte/Simple-Twitter-X-AIbot.git
   cd Simple-Twitter-X-AIbot
   ```

2. **Создайте и активируйте виртуальное окружение:**
   ```bash
   python -m venv venv
   source venv/bin/activate  # Для Windows: venv\Scripts\activate
   ```

3. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

## Настройка

### 1. Переменные окружения

Создайте файл настроек на базе шаблона:
```bash
cp .env.example .env
```

Откройте `.env` и впишите необходимые ключи:
```ini
LLM_API_KEY=your_llm_api_key_here
GEMINI_API_KEY=your_gemini_api_key_here # Опционально: для анализа графиков
TELEGRAM_BOT_TOKEN=your_telegram_bot_token
TELEGRAM_CHAT_ID=your_telegram_chat_id
TWITTER_CONSUMER_KEY=your_twitter_consumer_key
TWITTER_CONSUMER_SECRET=your_twitter_consumer_secret
TWITTER_ACCESS_TOKEN=your_twitter_access_token
TWITTER_ACCESS_TOKEN_SECRET=your_twitter_access_token_secret
TWITTER_BEARER_TOKEN=your_twitter_bearer_token
```

### 2. Настройки бота (`config.yaml`)

Откройте файл настроек `config.yaml` для изменения основных параметров:
```yaml
bot:
  language: "ru"              # Язык интерфейса ("en" или "ru")
  niche: "Искусственный Интеллект" # Ниша поиска контента
  tone_of_voice: "Экспертный блогер, профессионально и кратко"
  mode: "auto"                # Поиск источников: "auto" или "manual"
  max_length: 280             # Лимит длины твита (25000 для X Premium)
  interval: [3600, 10800]     # Интервал проверки в секундах (мин, макс)

llm:
  base_url: "https://integrate.api.nvidia.com/v1"
  model_name: "deepseek-ai/deepseek-v4-pro"
```

## Telegram Команды и Использование

Запустите основное приложение:
```bash
python main.py
```

Откройте диалог с ботом в Telegram. Вы можете использовать кнопки быстрого меню или выполнять следующие текстовые команды:

| Команда | Аргументы | Описание |
| :--- | :--- | :--- |
| `/start` | | Инициализирует бота и выводит главное меню с кнопками. |
| `/status` | | Показывает текущий статус, кулдауны, выбранную нишу и статистику. |
| `/parse` | | Запускает принудительную внеочередную проверку источников новостей. |
| `/queue` | | Открывает интерактивную очередь постов на утверждение. |
| `/history` | | Отображает заголовки 5 последних опубликованных постов. |
| `/noaccept` | | Переключает режим автоматического постинга без подтверждения. |
| `/set_lang` | `<en\|ru>` | Меняет язык интерфейса бота и генерируемых твитов. |
| `/set_limit` | `<number>` | Задает максимальный лимит символов для твита (например, 280 или 25000). |
| `/set_interval` | `<min> [max]` | Задает интервал проверки (например, `1h 3h` или `2h`). |
| `/set_niche` | `<name>` | Меняет тематическую нишу на лету. |
| `/set_mode` | `<auto\|manual>` | Переключает режим поиска источников контента. |
| `/set_tone` | `<tone>` | Меняет стиль написания текстов нейросетью. |
| `/add_rss` | `<url>` | Добавляет кастомную RSS-ленту (для manual-режима). |
| `/remove_rss` | `<url>` | Удаляет кастомную RSS-ленту. |
| `/add_sub` | `<name>` | Добавляет кастомный сабреддит (для manual-режима). |
| `/remove_sub` | `<name>` | Удаляет кастомный сабреддит. |
| `/sources` | | Показывает список текущих RSS-лент и сабреддитов. |
| `/clear_sources` | | Очищает сохраненный кэш автоматически найденных источников. |
| `/stop` | | Приостанавливает выполнение проверок по таймеру. |
| `/help` | | Выводит справочную информацию и список всех команд. |

## Архитектурные особенности

- **Асинхронность и отзывчивость:** Тяжелые операции сетевых запросов и взаимодействия с LLM выполняются через `asyncio.to_thread`. Это гарантирует, что Telegram-бот всегда мгновенно отвечает на действия пользователя и не зависает.
- **Надежность базы данных:** SQLite работает в режиме Write-Ahead Logging (WAL), поддерживая безопасную параллельную запись транзакций. Бот надежно защищен от публикации дубликатов.
- **Отказоустойчивость:** Автоматическое экранирование спецсимволов защищает от падений API Telegram при рендеринге HTML. Алгоритм умного усечения текста твита по предложениям сохраняет структуру мысли при превышении лимитов. Встроенный экспоненциальный бэкофф защищает от блокировок Reddit при получении ошибки `429 Too Many Requests`.
